  • The Wingman Dating app

    Date: 24/04/2019
    Grade: 8.2

  • For a project of my education, I developed a dating application together with my project group, which besides dating is also involved in implementation. For example, the application can be used to find people with the same hobbies. A buddy system is also used for this. Which is a built-in AI, which helps the user in making choices and taking steps, in order to create the best possible match and result.

    For this project I delivered several works with my project group, which is divided into a research report and a design report. The research report contains the competition analysis, Benchmark, research target group, in-depth interviews, personas, empathy map, User needs and concept scenarios.

    The design report contains the user flow, sitemap, customer journey, style guide, patterns & design principles, wireframes, system requirements, usability test and the hi-fi prototype.

  • Issue

    There are few differences between available dating applications when matching users, as users are often judged on their photos. There is insufficient insight into a new way of dating, which focuses on aspects other than visual material. There is also insufficient insight into the functional possibilities of sensors in a smartphone, this also plays a role in the research.

  • Objective

    An objective has been formulated based on the problem definition. This objective serves as a guideline within the project so that it is clear to everyone who is working towards it.

    The objective formulated on the basis of the problem statement is as follows: “Providing insight into new ways to match users with the same interests. This takes into account the functional possibilities of sensors in a smartphone, with regard to the experience of users of a dating application. These sensors must contribute to the match process. ”

  • Research Question

    A research question has been formulated based on the problem definition and the objective.

    The research question formulated on the basis of the problem definition and objective goes as follows: “What are the functional possibilities for the target group while using a dating application with a focus on the experience of the users.”

    The target group consists of both men and women aged 18 to 25 who are looking for users with the same interests.

  • Conclusion

    The app’s prototype represents the research and answers the main question. An application with more functional possibilities has been made for the target group with the focus on the perception of the users. The application uses location and artificial intelligence sensors. The end product has achieved the objective of this project.
